# 🏡 UrbanNest

UrbanNest is a full-stack web application inspired by Airbnb that allows users to browse, create, edit, and manage property listings.

## 🚀 Live Demo

🔗 **Live Website:** [UrbanNest Live](https://urbannest-whx7.onrender.com)

---

## ✨ Features

* 🏠 View all property listings
* ➕ Create new listings
* ✏️ Edit existing listings
* 🗑️ Delete listings
* 📱 Responsive user interface
* 🗄️ MongoDB Atlas cloud database integration

## 📦 Installation & Setup

### Clone the repository

```bash
git clone https://github.com/YashRajveerShahi/UrbanNest.git
cd UrbanNest
```

### Install dependencies

```bash
npm install
```

### Create a `.env` file in the root directory and add:

```env
ATLASDB_URL=your_mongodb_atlas_connection_string
```

### Run the application

```bash
npm start
```

The application will run on:

```bash
http://localhost:8080
```
